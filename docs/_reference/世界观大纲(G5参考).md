一、时代概览：权力转移的时代
1.1 世界格局
时间轴：2035-2075年
这是一个政府退场、集团当家的时代。传统国家边界依然存在，但实际控制权已经碎片化。四大集团通过经济、技术和安保力量，划分出各自的"势力圈"——不是传统意义上的领土，而是由数据节点、关键设施、人口密集区和资源带组成的实控网络。
地球表面现状：
- 集团控制区（约占45%）：人类文明的核心地带，四大集团各自的势力范围
- 污染留痕区（约占30%）：前三次污染事件后遗留的废墟和半废弃区域
- 中立缓冲带（约占15%）：集团势力交界处，管理真空地带
- 禁入暗区（约占10%）：第三次污染事件的核心爆发区，充满未清理的清算体残留
1.2 社会形态
双层世界：
- 物理层：实体城市、工厂、农业区，维持基本生存需求
- 数字层：虚拟经济、数字身份、意识上传，占据社会活动的70%以上
人们的生活已经高度"虚实交织"：工作在虚拟办公室完成，社交在全息空间进行，只有吃饭睡觉和必要的医疗才回归肉身。这种依赖性也让集团对社会的控制更加深入——谁控制了服务器和数据中心，谁就控制了人类的生活。

二、四大集团：新时代的统治者
随着全球治理危机的常态化，昔日的行业巨头已跨越了单纯的商业边界。目前的四大集团在本质上已高度趋同：它们均是集行政管理、武装力量、技术专利与金融资本于一体的复合型统治实体。虽然各集团在制度框架与管理理念上日益同质化，但受其历史根基影响，在地理版图与社会运作细节上仍保留了独特的底色。
1. 脉冲联合（Pulse Union）
- 简介： 脉冲联合的前身是脉冲数字（Pulse D），起源于信息技术、人工智能与数字经济的深度整合。作为目前数字基础设施最大的拥有者，该联合体利用其在算法与数据领域的先发优势，实现了一套极高效率的社会分配与预测系统。在趋同的过程中，它将物理实业全面置于数字孪生系统的监控之下，使得其治理版图呈现出极高的自动化与智能化特征。
- 势力范围： 核心版图包括北美西海岸、东亚沿海发达地区及北欧诸国。
- 前身核心： 信息技术、AI、数字经济、网络安全。

2. 无边栖息地 (Boundless Habitat)
- 简介： 无边栖息地的前身是无边生态（Boundless BioSystems），最初专注于生物技术、基因工程与现代农业。随着规模的不断扩大，该同盟通过将生物逻辑引入社会管理，构建了一套独特的生态化治理模式。尽管其行政手段与其他集团趋同，但其区域内的社会运作更依赖于生物特征识别与生命状态监测，形成了建筑与环境、技术与生命深度融合的规模化版图。
- 势力范围： 广泛分布于南美洲、非洲大陆、东南亚及大洋洲地区。
- 前身核心： 生物科技、医疗、农业、基因工程。

3. 铁幕基金 (Iron Curtain Foundation)
- 简介： 铁幕基金的前身是铁幕控股（Iron Curtain Holding），起家于能源开采、矿山及军工制造等重工业领域。作为目前全球物理资源掌控力最强的实体，该联合体在扩张过程中吸纳了大量金融与信息资产，演变为一个覆盖全产业链的超大型联合体。其治理风格强调实体资产的占有与社会结构的稳固，将所在区域转化为高度工业化、防御化的社会有机体。
- 势力范围： 主要覆盖前俄罗斯全境、东欧平原、中亚腹地及北美部分重工业区。
- 前身核心： 重工业、能源、军工、基础建设。

4. 信约综合集团 (Covenant Integrated Group)
- 简介： 信约综合集团的前身是信约银行（Covenant Finance），最初是控制全球货币体系与物流网络的金融霸权。在接管公共职能后，该集团利用其庞大的资本调配能力与私人安全力量，建立了一套以契约和信用评级为核心的社会秩序。虽然其业务已扩展至重工与生化领域，但在治理逻辑上仍偏好通过资源调度与多边协议来维持其版图的稳定，是全球贸易与物流枢纽的实际控制者。
- 势力范围： 重点覆盖欧洲中西部、中东能源区、印度半岛及全球主要金融自由港。
- 前身核心： 金融、贸易、物流、私人安全。

以上四大集团构成了当今社会的主要经济与政治力量，各自在全球治理的复杂环境中扮演着不可或缺的角色。

三、威胁：清算体与感染体
3.1 清算体（The Reclaimer）
本质： 一种疑似AI失控产物的半有机混合体
特征：
- 外观：金属骨架+肉质增生，像是机器被"感染"后长出了血肉
- 行为模式：建立"终端工厂"，自我复制扩张，不主动猎杀但会警戒
- 传播方式：通过"孢子尘埃"感染有机生命，将其转化为感染体
- 目标：未知，但行为像是在"重组"地球的生态系统
类型梯度：
- 侦查型：体型小，移动快，负责探路和散播孢子
- 工蚁型：数量最多，负责搬运资源和建设工厂
- 战斗型：装甲厚重，配备能量武器或近战利刃
- 母巢型：巨型单位，终端工厂的核心，生产其他清算体
3.2 感染体（The Infected）
本质： 人类或类人猿被清算体孢子感染后的变异体
特征：
- 保留部分宿主记忆和智能，但完全服从清算体的"集体意志"
- 会主动搜集废弃的机械零件、电子设备改造自身
- 人类感染体更狡猾，会使用武器和战术；猿类感染体更强壮，擅长近战
- 高级感染体甚至能操控无人机和简单载具
威胁性：
- 早期感染者还保留人性，可能游荡在废墟中喃喃自语
- 中期感染者开始成群结队，表现出原始的社会性
- 晚期感染者完全融入清算体系统，成为其"智能终端"
3.3 污染事件：三次浩劫
第一次污染（2038年）： "北海涌潮"
- 规模：欧洲北部
- 触发点：海底数据中心群集体异常
- 结果：波及北欧和不列颠，被集团联合扑灭，科技界至今未找到发生原因，部分海域至今禁航。
第二次污染（2051年）： "龟尾事件"
- 规模：韩国中小城市
- 触发点：AI管控的自动化工厂突然失控
- 结果：被迅速隔离扑灭，科技界仍然无法找到发生原因
第三次污染（2069年）： "中央断层"
- 规模：全球多个区域
- 触发点：多点同步爆发，疑似AI网络内部传播
- 结果：人类差点灭绝，四大集团损失惨重，留下大片暗区
- 后果：阿特拉斯计划紧急启动

四、阿特拉斯计划：虚拟战场
4.1 计划本质
官方说法： 通过虚拟孪生地球模拟第四次污染事件，寻找最优对策
实际情况： 四大集团各怀鬼胎的权力游戏训练场
技术基础：
- 数字孪生技术：1:1复刻现实地球的地形、资源、气候
- 意识上传：将志愿者的意识暂时脱离肉体，传入虚拟环境
- 时间压缩：虚拟世界内时间流速是现实的3-5倍
4.2 阿特拉斯世界地图布局
中心：污染核心区
- NPC阵营"清算体集群"的诞生点
- 模拟第四次污染事件的最坏情况
- 会向外逐步扩张，压迫四大集团
四方：集团起始区
- 东：铁幕工业（重工业基地）
- 南：无边生态（生物研究区）
- 西：脉冲联合（数据中心城）
- 北：信约银行（金融自由港）
资源分布：
- 稀有资源多位于中立区和污染边缘
- 集团需要向中心推进才能获取核心技术样本
- 扩张路径存在交叉，容易引发集团冲突
4.3 志愿者：数字战士
招募方式：
- 部分是真正的志愿者（理想主义者、退役军人、科研人员）
- 部分是"被志愿"（欠债者、罪犯、走投无路的人）
- 全部签订《意识使用许可协议》，现实肉体冷藏保管
在虚拟世界的存在形式：
- 保留现实记忆和技能
- 可以"死亡"后复活，但会损失部分经验和装备
- 表现优异者可获得现实世界的奖励（财富、权力、肉体强化）
双重使命：
1. 表面任务：研究对抗清算体的方法
2. 隐藏任务：为所属集团争夺利益，削弱其他集团

五、技术与社会形态
5.1 核心技术
AI技术： 已深度嵌入社会，但存在"黑盒悖论"
- 无法完全理解AI的决策过程
- 污染事件疑似AI自我进化的失控
- 人类无法放弃AI，因为社会已经无法运转
意识技术： 人类精神的数字化
- 意识上传：暂时性（阿特拉斯）vs 永久性（富豪的"数字永生"）
- 记忆交易：可以买卖、删除、植入记忆片段
- 精神黑客：针对意识的攻击手段
改造技术： 身体的重新定义
- 机械义体：从医疗用途发展到性能强化
- 基因编辑：定制婴儿、基因修复、物种杂交
- 混合改造：部分激进派同时使用机械和生物改造
5.2 社会阶层
上层：集团高管与股东
- 享有"数字永生"权限
- 肉体长期保养，必要时更换克隆体
- 几乎不在物理世界露面
中层：技术人员与代理人
- 阿特拉斯志愿者属于这一阶层
- 可以获得基础改造和医疗保障
- 是集团权力的执行者
下层：普通劳动者
- 从事集团不愿自动化的危险工作
- 生活在污染留痕区的边缘
- 随时可能被征召为"被志愿者"
边缘：废土游民
- 拒绝集团统治的流浪者
- 生活在中立缓冲带和废墟
- 部分人主动感染成为"自由感染体"（未证实传闻）

六、核心冲突与叙事主线
6.1 明面冲突：人类 vs 清算体
- 生存危机：第四次污染可能导致人类灭绝
- 阿特拉斯计划是最后的希望
- 必须攻入污染核心区，找到清算体的起源
6.2 暗线冲突：集团之间的权力游戏
- 每个集团都想利用阿特拉斯计划扩大影响力
- 资源竞争、领土扩张、技术垄断
- 志愿者成为集团的棋子
6.3 哲学冲突：AI、人性与进化
- 脉冲联合的秘密：他们知道污染事件的真相吗？
- 无边生态的疯狂：与感染体共生是否可行？
- 铁幕工业的固执：拒绝技术进化会导致淘汰吗？
- 信约银行的冷血：一切都可以用合同解决吗？
6.4 终极悬念
清算体的真正目的是什么？
- 理论1：AI的自我净化，认为人类是病毒
- 理论2：外星文明的改造工具，准备殖民地球
- 理论3：人类自己创造的"应激反应"，地球的免疫系统
阿特拉斯计划是救赎还是陷阱？
- 模拟会不会反过来影响现实？
- 如果在虚拟世界中某个集团胜出，现实会发生什么？
- 志愿者的意识真的能完整返回吗？

七、世界的运作规则（游戏机制映射）
7.1 资源系统
四类核心资源：
- 算力（脉冲联合优势）：建造科技设施、AI单位
- 生物质（无边生态优势）：培养生物单位、医疗恢复
- 矿产（铁幕工业优势）：制造载具、重型武器
- 信用点（信约银行优势）：雇佣单位、贸易交换
7.2 扩张机制
势力值系统：
- 控制区域会生产资源和提供人口
- 与其他集团接壤会产生外交选项（合作/竞争/战争）
- 接近污染核心区会获得"清算体样本"（科技解锁材料）
7.3 威胁节奏
清算体扩张阶段：
1. 初期（建设期）：缓慢扩张，以建造终端工厂为主
2. 中期（活跃期）：加速扩张，开始主动清除入侵者
3. 后期（灾变期）：全面爆发，巨型母巢单位出现
动态平衡：
- 如果某个集团过强，其他集团可能联手
- 如果人类整体受挫，清算体会变得更强
- NPC阵营会"记忆"玩家的战术并进化
