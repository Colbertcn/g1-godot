# 世界观大纲: 第一次污染事件 (G1 Project)

## 概述

本作 (G1) 发生于 G5 战略游戏故事线之前，聚焦于人类文明与"清算体"的首次大规模冲突——**第一次污染事件**（First Contamination Event，2038年）。

玩家将扮演一名精锐机甲驾驶员，亲历这场突如其来的灾难，见证旧世界的崩塌与四大集团崛起的序幕。

## 时间点：2038年 "北海涌潮"

*   **背景**: 2038年，全球经济处于高度自动化转型的阵痛期。各大科技巨头（后来的四大集团前身）掌握了核心算力与资源，但传统国家政权依然存在。
*   **事件爆发**: 北海（North Sea）海底数据中心群突发原因不明的集体异常。紧接着，大量半有机机械生物——"清算体"从海底涌出，迅速袭击了沿岸城市与海上设施。
*   **初期反应**: 各国常规军队在面对能够侵蚀、同化机械装备的清算体时节节败退。
*   **转折点**: 私人安保公司与科技巨头紧急投入了实验性的**机动装甲（机甲）**。这些机甲装备了针对性的能量护盾与高机动推进器，成为对抗清算体潮汐的唯一希望。

## 集团设定 (早期形态)

1.  **地球防卫联合 (EDC - Earth Defense Coalition)**
    *   代表传统国家力量的松散联盟，指挥混乱，装备量大但落后。
    *   玩家初期名义上的所属方。

2.  **先锋科技 (Vanguard Tech - 脉冲联合前身)**
    *   提供最先进的火控系统与无人机支援。
    *   即使在危机中也在秘密收集实战数据。

3.  **生物圈公司 (Biosphere Inc. - 无边栖息地前身)**
    *   研究清算体的生物特性，提供抗腐蚀涂层与应急医疗。
    *   对感染体的态度暧昧不明。

4.  **重工联盟 (Heavy Industries Alliance - 铁幕基金前身)**
    *   提供重型装甲与大口径实弹武器。
    *   主张以核武等极端手段彻底清洗污染区。

5.  **清算体 (The Reclaimer)**
    *   **起源**: 未知。疑似深海数据中心觉醒的失控AI与海底生物的融合。
    *   **形态**: 早期清算体带有明显的海洋生物特征（甲壳、触须）与服务器机架、线缆的混合风格。
    *   **感染体**: 被孢子寄生的人类或动物，丧失理智，攻击性极强。

## 玩家角色

*   **身份**: 前空军王牌飞行员，被紧急征召加入"三叉戟"机甲特遣队。
*   **任务**:
    1.  **拦截**: 阻止清算体登陆主要人口密集区。
    2.  **救援**: 掩护平民与重要科研人员撤离。
    3.  **调查**: 深入污染源头（北海中心），寻找灾难爆发的真相。

## 核心冲突

*   **生存**: 在无尽的兽潮中活下去。
*   **背叛**: 发现某些巨头公司在利用灾难清洗异己，甚至主动诱发污染扩散。
*   **抉择**: 服从腐朽的防卫联合命令，还是与新兴的集团力量合作？
