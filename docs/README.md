# 项目文档说明

本文档目录存放了 G1-project 的所有设计文档、参考资料和相关说明。

## 目录结构

```
docs/
├── Design/              # 游戏核心设计文档
│   ├── 设计核心.md      # 游戏系统架构与详细设计文档 (当前主要参考)
│   ├── Todo.md          # 待办事项与任务优先级
│   └── 世界观大纲.md    # 游戏剧情、世界观、势力设定及背景故事
├── _reference/          # 参考资料归档
│   └── kiro/            # 外部参考项目或文档
└── README.md            # 本文件
```

## 关键文件说明

### 1. [设计核心.md](Design/设计核心.md)
**用途**: 游戏的工程实现蓝图。
**内容**:
- **概述**: 游戏类型（2D 载具射击）、背景设定简介。
- **架构**: 移动系统、经济系统、武器与僚机系统。
- **组件与接口**: 核心属性定义（生命值、护盾、经验值等）。
- **数据模型**: 武器类型（中庸、散弹、贯通）。

### 2. [世界观大纲.md](Design/世界观大纲.md)
**用途**: G1项目的核心剧情大纲。
**内容**:
- **背景**: 第一次污染事件（2038年）。
- **势力**: 地球防卫联合与四大集团的前身。
- **敌人**: 早期清算体与感染体。

### 3. [世界观大纲(G5参考).md](Design/世界观大纲(G5参考).md)
**用途**: 原始世界观设定（来源于相同IP的另一个游戏项目：G5），作为G1的未来时间线参考。
**内容**:
- **时代概览**: 2035-2075年的世界格局。
- **四大集团**: 成熟期的集团设定。
- **阿特拉斯计划**: G5的主要背景。

### 4. [关卡路线规划](Design/LevelDesign.md)
**用途**: 定义游戏战役流程与环境设定。
**内容**: 7个阶段性关卡，从汉堡工业区到北海数据中心。



## 使用指南
- 进行**代码开发**时，请优先参考 `设计核心.md` 中的架构与接口定义。
- 进行**美术资源制作**或**剧情脚本编写**时，请深入阅读 `世界观大纲.md`。
- `_reference/` 目录下的文件仅作参考，不作为本项目开发的直接依据，除非在 `设计核心.md` 中有明确引用。
